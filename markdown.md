## 마크다운 문법 살펴보기

### < 헤딩 >

```
# 헤딩1
## 헤딩2
### 헤딩3
#### 헤딩4
##### 헤딩5
###### 헤딩6
```

<결과>

# 헤딩1

## 헤딩2

### 헤딩3

#### 헤딩4

##### 헤딩5

###### 헤딩6

<br>

### < 강조 및 이탤릭체 >

```
_이탤릭체_

**굵게**

**_강조 및 이탤릭체 혼합_**
```

### < 리스트 >

#### 번호가 있는 리스트

```
1. 내용
2. 내용
```

#### 번호가 없는 리스트

```
* 1뎁스
  * 2뎁스 (tap)
```

<결과>

1. 내용
2. 내용

- 1뎁스
  - 2뎁스
